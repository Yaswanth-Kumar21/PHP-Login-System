# 🔐 PHP Login System

## 📌 Project Overview

This project is a simple **PHP Login System** developed as part of my internship at SNS Infotech Innovations.
It authenticates users by verifying their credentials using a MySQL database.

---

## 🚀 Features

* User login authentication
* Secure password verification
* Database connectivity using MySQL
* Prepared statements to prevent SQL injection
* Clean and responsive UI
* Error handling for invalid login

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS
* Backend: PHP
* Database: MySQL
* Server: XAMPP
* Editor: VS Code

---

## ⚙️ How It Works

1. User enters **User ID and Password**
2. Data is sent to the server using POST method
3. PHP processes the request
4. MySQL database checks credentials
5. System returns:

   * ✅ Login Successful
   * ❌ Invalid User ID or Password

---

## 🗄️ Database Structure

Table: `users`

| Column   | Type    |
| -------- | ------- |
| id       | INT     |
| userId   | VARCHAR |
| password | VARCHAR |

## ▶️ How to Run

1. Install XAMPP
2. Start Apache & MySQL
3. Place project folder in `htdocs`
4. Import database in phpMyAdmin
5. Run in browser:

   ```
   http://localhost/your-folder-name
   ```
